> tsconfig.tsbuildinfo This file stores:
                    which files were compiled
                    dependency graph
                    timestamps
                    hashes
                    compiler state

                    so next build can reuse previous work.

Without it every build compile ALL files again

With .tsbuildinfo only changed files are recompiled

This makes builds MUCH faster in large projects.

EXAMPLE:
Suppose you have:
100 TypeScript files

You change only:
Navbar.tsx

Without .tsbuildinfo:
TypeScript recompiles everything.

With .tsbuildinfo:
TypeScript recompiles only affected files.
